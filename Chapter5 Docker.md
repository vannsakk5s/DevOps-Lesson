# Chapter 5 — Docker

## Meaning
Docker គឺជា container platform។
Container = lightweight application environment។

## Example
- **App**
- **Dependencies**
- **Libraries**

All inside a container.

## Commands
- **Pull image:** `docker pull nginx`
- **Run container:** `docker run -d -p 80:80 nginx`
- **Show containers:** `docker ps`
- **Stop container:** `docker stop container_id`

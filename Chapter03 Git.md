# Chapter 3 — Git

## Meaning
Git គឺជា Version Control System។
វាជួយ track code changes។

**Example:**
- version 1
- version 2
- version 3

## Commands
- **Initialize repo:** `git init`
- **Add files:** `git add .`
- **Commit changes:** `git commit -m "update code"`
- **Push to remote:** `git push`
- **Pull changes:** `git pull`

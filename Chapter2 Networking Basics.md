# Chapter 2 — Networking Basics

## Meaning
Networking គឺជាវិធីដែល computers communicate ជាមួយគ្នា។

**Example:**
`Browser → Server → Response`

## Important Concepts
- **IP Address**
- **DNS**
- **Port**
- **HTTP**
- **HTTPS**
- **SSH**

## Example
`http://3.26.183.226`

- **3.26.183.226** = IP Address
- **http** = Protocol

## Commands
- **Ping server:** `ping google.com`
- **Check website:** `curl google.com`
- **Check open ports:** `netstat -tulnp`
